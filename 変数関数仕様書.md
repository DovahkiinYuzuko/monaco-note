# 変数・関数仕様書

## フロントエンド (src/components)

### MonacoEditor (Editor.tsx)
Monaco EditorをReactコンポーネントとしてラップ。エディタの設定とイベントハンドリングを担当。
- **Props**:
  - alue: エディタに表示するテキスト
  - onChange: 内容変更時のコールバック
  - ontSize, ontFamily: フォント設定
  - 	heme: 'vs-dark' | 'light'
- **Internal Logic**:
  - egisterCompletionItemProvider: カスタムスニペットの登録
  - utomaticLayout: true: ウィンドウリサイズへの追従

### Preview (Preview.tsx)
MarkdownをHTMLに変換して表示。
- **Library**: marked + highlight.js
- **Security**: DOMPurify によるサニタイズ
- **Sync Scroll**: エディタとのスクロール同期

### Settings (Settings.tsx)
アプリの設定変更用モーダル。
- AppConfig: 設定値の管理。
- handleRefreshFonts: バックエンドの efresh_os_fonts を呼び出し、システムフォント一覧を更新。結果は localStorage にキャッシュされる。
- initFonts: アプリ起動時または設定展開時に get_os_fonts を呼び出し、フォント一覧を初期化。
- **Dependencies**:
  - invoke('refresh_os_fonts'): システムフォントの再スキャン（Rust）
  - invoke('get_os_fonts'): キャッシュされたフォント一覧の取得（Rust）

## フック (src/hooks)
- useConfig.ts: 設定の永続化管理 (config.json)
- useTabs.ts: タブ管理とファイル操作の連動
- useAutoSave.ts: 一定時間ごとの自動保存

## バックエンド (src-tauri/src)
- lib.rs / main.rs: Tauriのセットアップとコマンド登録
- ile_ops.rs: ファイルの読み書き、文字コード変換 (encoding_rs)
- ont_ops.rs: システムフォントの列挙とキャッシュ (onts_cache.json)
